BibYAML

    BibYAML - stands for Bibliographic YAML, and aims to be a new standard for managing, formatting and storing bibliographies. A BibTEX to BibYAML converter can be found at bibyaml.appspot.com.

Pycoevol

    Pycoevol - A Python workflow to study protein-protein coevolution and interaction, is a set of tools for inter-protein coevolution and interaction. Several coevolution measures and several processing tools are implemented. A GUI in delphi is also provided.

Blocks

    Blocks - Multiple Sequence Alignment (MSA) processing tool, is a small tool for processing MSA that gets gappy or conserved columns. It scores alignments using the classic Sum-of-pairs function and converts MSA in FASTA to MSF.

PyCMA

    PyCMA - A python module for correlated mutation analysis, is a python module that implements a wide range of correlated mutation methods. Preprocessing methods, as well as several facilities for analysis of coevolution scores are provided. (to be deployed soon)
